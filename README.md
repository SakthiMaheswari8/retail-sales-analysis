Retail Sales Analysis – SQL Project
 
------ Project Overview ------

•	This project is a beginner-level SQL data analysis project created as part of my learning journey in Data Analytics.

•	The dataset represents retail sales transactions, and the goal of this project is to analyze the data using SQL to generate meaningful business insights such as sales trends, customer behavior, and product performance.

•	I started with a basic project structure and extended it by adding additional SQL queries and business analysis to improve my understanding and skills.

 ------ Objectives ------
 
•	Create and manage a retail sales database

•	Perform data cleaning by removing missing values

•	Conduct exploratory data analysis (EDA)

•	Write SQL queries to solve real-world business problems

•	Generate insights from sales and customer data

------ Tools Used ------

•	MySQL (SQL)

•	GitHub (for project hosting)
